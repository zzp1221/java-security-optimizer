# Java 模块学习（第三版）：方法级打卡路线

第三版目标：把“能看懂模块”升级为“能逐方法定位、能独立改造”。

## 你会得到什么

- 每个模块的“类 -> 方法”必读清单
- 每个方法的输入、输出、关键副作用
- 典型陷阱与改造入口
- 可在 IDE 中逐项打卡的学习路径

## 文档目录（第三版）

1. `v3-01-analysis-方法级清单.md`
2. `v3-02-task-storage-方法级清单.md`
3. `v3-03-plugin-rulepack-security-方法级清单.md`
4. `v3-04-autofix-governance-方法级清单.md`
5. `v3-05-30天学习计划.md`

## 使用方式

1. 打开对应模块文档。  
2. 按“必读方法”顺序在 IDE 中跳转。  
3. 每看完一个方法，回答文档里的自测问题。  
4. 每章完成 1 个小改造并补 1 个测试。  

## 学习节奏建议

- 每天 60-90 分钟
- 前 30 分钟读方法
- 中间 20 分钟画调用链
- 后 20-40 分钟做小改造和测试
