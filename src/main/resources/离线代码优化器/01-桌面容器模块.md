# 01-桌面容器模块（Tauri）

## 项目总体是做什么
项目是一款离线桌面代码优化工具：导入本地代码，执行本地分析，输出可落地的优化建议与修复方案。当前主语言是 Java，并预留 C++ 扩展。

## 本模块要干什么
- 提供跨平台桌面壳，承载前端界面。
- 管理文件系统访问授权（选择项目目录、读取源码、写入报告）。
- 拉起并管理 Java 分析 sidecar 进程（启动、健康检查、停止、重启）。
- 作为本地网关转发任务请求和进度事件，避免前端直接访问引擎。

## 输入与输出
- 输入：前端发起的分析任务、规则包导入请求、修复应用请求。
- 输出：任务状态流、分析结果、错误信息、修复执行结果。

## 第一批交付物
- Tauri 命令：`selectProjectDir`、`startEngine`、`submitTask`、`cancelTask`。
- sidecar 生命周期管理器（单实例 + 自动重启）。
- 权限白名单配置（仅允许用户授权目录访问）。

## 如何开始
- 初始化 `apps/desktop`，配置 Tauri v2 与必要插件（shell/fs/dialog/process）。
- 定义 Rust Command 与前端 IPC 协议映射。
- 打通最小链路：前端按钮 -> Rust 命令 -> Java 健康检查接口。
