# 离线代码优化桌面应用-项目总览

## 项目总体是做什么
本项目要构建一款**离线桌面代码分析工具**。用户将本地代码仓导入应用后，不依赖云端即可完成代码质量分析、性能优化建议识别、规则命中展示与自动修复预览。第一阶段聚焦 Java，架构上预留 C++ 插件位。

核心目标如下：
- 离线可用：在内网/断网环境下持续工作。
- 大仓库可分析：支持大文件与大规模项目的增量分析。
- 规则可扩展：支持内置规则和团队自定义规则包。
- 结果可执行：不仅提示问题，还支持安全级别分层的自动修复建议。

## 核心技术栈
- 桌面容器：Tauri v2（Rust）
- 前端：TypeScript + React + Monaco Editor
- 分析引擎：Java 21 + JavaParser
- 本地数据：IndexedDB（索引）+ 文件系统（大报告）
- 规则共享：规则包 Manifest + SHA-256 + 签名验签

## 模块划分
- 01：桌面容器与进程编排
- 02：前端工作台与交互
- 03：Java 分析引擎
- 04：规则中心与规则包
- 05：数据存储与索引
- 06：安全与签名体系
- 07：任务调度与可观测
- 08：插件扩展与 C++ 预留

## 里程碑建议
- M1：导入项目 -> 发起分析 -> 展示问题（最小闭环）
- M2：20 条 Java 规则 + 大仓库增量分析
- M3：自动修复预览/应用/回滚
- M4：规则包签名、团队共享、审计能力
